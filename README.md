# How can I install it?

Probably just drag the `Space.xccolortheme` to your `~/Library/Developer/Xcode/UserData/FontAndColorThemes/`

# Font?

It's [Space Mono](https://fonts.google.com/specimen/Space+Mono?category=Monospace)
